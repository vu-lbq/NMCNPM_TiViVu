Project Folder Structure

Root
- LICENSE
- README.md
- requirement.txt
- project-structure.txt
- apps/                                 Thư mục chính của dự án
	- backend/
		- package.json                  : Phụ thuộc backend, script (build/start), thông tin dự án
		- pnpm-lock.yaml                : Tệp khóa đảm bảo phiên bản gói chính xác
		- README.md                     : Tài liệu mô tả dịch vụ backend
		// Backend Detailed
		- .env                          : Biến môi trường (DB URL, JWT, Mailjet)
		- .sequelizerc                  : Cấu hình đường dẫn cho Sequelize CLI (models/migrations/seeders)
		- createdb.js                   : Script tạo/khởi tạo cơ sở dữ liệu
		- server.js                     : Khởi động máy chủ Express
		- src/
			- app.js                      : Thiết lập ứng dụng Express (middleware, CORS, router)
			- dbInit.js                   : Logic khởi tạo/đồng bộ cơ sở dữ liệu
			- config/
				- config.json               : Cấu hình môi trường cho Sequelize
				- db.js                     : Kết nối instance Sequelize
			- controllers/
				- adminController.js        : Tác vụ quản trị (người dùng/vai trò/bảng điều khiển)
				- aiController.js           : Endpoint chat AI/tích hợp
				- authController.js         : Đăng ký/đăng nhập/thông tin người dùng; cấp token
				- controller.js             : Controller mẫu/dùng chung
				- conversationController.js : CRUD hội thoại/danh sách
				- feedbackController.js     : Gửi/xem/quản lý phản hồi
				- messageController.js      : CRUD tin nhắn/danh sách theo hội thoại
				- nlpController.js          : Tiện ích NLP (dịch/tóm tắt)
				- passwordController.js     : Quên/đặt lại mật khẩu (Mailjet)
				- vocabController.js        : CRUD từ vựng; tự động lấy phiên âm
				- voiceController.js        : Endpoint chuyển văn bản thành giọng nói/STT/voice-chat
			- middlewares/
				- auth.js                   : Bảo vệ xác thực bằng JWT (đảm bảo người dùng/admin)
				- errorHandler.js           : Middleware định dạng lỗi
			- migrations/
				- 20251203000100-create-feedbacks.js           : Tạo bảng feedbacks
				- 20251203001000-add-role-to-users.js          : Thêm cột vai trò vào bảng users
				- 20251203002000-add-phonetics-to-vocabularies.js : Thêm phiên âm cho vocabularies
			- models/
				- Conversation.js           : Model hội thoại
				- Feedback.js               : Model phản hồi
				- index.js                  : Khởi tạo các model/liên kết
				- Message.js                : Model tin nhắn
				- User.js                   : Model người dùng (username/passwordHash/role)
				- Vocabulary.js             : Model từ vựng (word/phonetics)
			- routes/
				- conversationRouter.js     : Endpoint hội thoại/tin nhắn
				- mainRouter.js             : Gắn kết router gốc cho API
			- seeders/
				- initall.js                : Seed dữ liệu ban đầu (admin/người dùng)
			- services/
				- aiService.js              : Lớp bọc LLM (prompt/response)
				- mailService.js            : Gửi email qua Mailjet
				- sttService.js             : Tích hợp chuyển giọng nói thành văn bản (STT)
				- ttsService.js             : Tích hợp chuyển văn bản thành giọng nói (TTS)
			- utils/
				- apiError.js               : Lớp lỗi API tiêu chuẩn
				- apiResponse.js            : Trợ giúp trả về thành công
				- resetToken.js             : Tạo/xác thực token đặt lại mật khẩu
---------------------------------------------------------------------------------------------------------
	- frontend/
		- index.html                    : Khung HTML Vite; gắn React tại `#root`
		- package.json                  : Phụ thuộc frontend và script Vite
		- pnpm-lock.yaml                : Tệp khóa cho các gói frontend
		- README.md                     : Tài liệu ứng dụng frontend
		- public/
			- images/
				- favicon.svg               : Biểu tượng trang phục vụ từ `/images/favicon.svg`
		- src/
			- App.jsx                     : Thành phần ứng dụng: Router, các route, kết nối auth provider
			- main.jsx                    : Khởi động React/Vite: tạo root và render `<App/>`
			- layouts/
				- MainLayout.jsx            : Layout dùng chung (header/sidebar/nội dung)
			- pages/
				- LandingPage.jsx           : Trang giới thiệu/điểm vào; liên kết đăng nhập/đăng ký
				- LoginPage.jsx             : Form đăng nhập/đăng ký; tuỳ chọn Ghi nhớ đăng nhập; điều hướng
				- ForgotPasswordPage.jsx    : Yêu cầu đặt lại mật khẩu qua email
				- ResetPasswordPage.jsx     : Đặt lại mật khẩu bằng token
				- ChatPage.jsx              : Giao diện chat chính; điều khiển STT/TTS; danh sách hội thoại
				- admin/
					- AdminDashboard.jsx      : Tổng quan admin; liên kết nhanh/thống kê
					- AdminUsers.jsx          : Quản lý người dùng (danh sách, vai trò); thao tác dành cho admin
					- AdminFeedback.jsx       : Xem xét phản hồi người dùng
			- components/
				- MessageBubble.jsx         : Hiển thị tin nhắn chat với kiểu dáng và hành động
				- VoiceChatModal.jsx        : Modal thoại-đến-thoại; ghi âm và phát hồi đáp AI
				- DictionaryModal.jsx       : Tra cứu từ; định nghĩa/cách dùng
				- VocabularyModal.jsx       : Từ vựng đã lưu; phiên âm + nút phát âm
				- FeedbackModal.jsx         : Thu thập phản hồi người dùng; gửi lên backend
				- Hero.jsx                  : Landing hero section with CTA and branding
				- Carousel.jsx              : Auto-advancing image carousel for landing
				- FeatureTrio.jsx           : Three feature cards highlighting capabilities
			- hooks/
				- useSpeech.jsx             : Hỗ trợ nhận dạng giọng nói và tổng hợp giọng nói trên trình duyệt
			- context/
				- AuthProvider.jsx          : Trạng thái auth, Ghi nhớ đăng nhập, bootstrapping, đăng nhập/đăng ký/đăng xuất
				- authContext.js            : Đối tượng React context cho trạng thái auth
				- useAuth.js                : Hook truy cập context auth trong component
			- services/
				- api.jsx                   : Client API: auth/chat/voice/admin/vocab; đính kèm JWT
			- utils/
				- sttSanitize.js            : Làm sạch kết quả STT (gộp các lặp lại tức thời)
				- docs/
						- architecture_mermaid.mmd        : Sơ đồ kiến trúc hệ thống (Mermaid)
						- architecture.puml               : Sơ đồ kiến trúc hệ thống (PlantUML)
						- class-diagram-mermaid.mmd       : Sơ đồ lớp (Mermaid)
						- class-diagram.puml              : Sơ đồ lớp (PlantUML)
						- usecase-en.md                   : Mô tả tình huống sử dụng bằng tiếng Anh
						- usecase-mermaid.mmd             : Sơ đồ trường hợp sử dụng (Mermaid)
						- usecase-vi.md                   : Mô tả tình huống sử dụng bằng tiếng Việt
						- folder_structure.txt            : Cấu trúc thư mục dự án và chú thích tệp
				- scripts/                          : Script hỗ trợ phát triển/tự động hóa
					- Đối tượng React context cho trạng thái xác thực
